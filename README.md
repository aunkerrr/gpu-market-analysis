# GPU Market Analysis
A simple tool for analyzing GPU market data from CSV files, built with Java.

## Description
This project aims to provide a simple tool for analyzing GPU market data from CSV files.

## Features
- **CSV Parsing**: Reads and processes GPU data from specified CSV files.
- **Data Analysis**: Performs various analyses on the GPU data, such as price-to-performance ratios, market trends, and more.
- **Reporting**: Generates reports or visualizations based on the analysis.

## Technologies Used
- Java
- Apache Commons CSV
